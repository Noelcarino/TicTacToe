# Tic Tac Toe (Currently at MVP)

A React Application where you can play Tic Tac Toe

### Features
    1) User can play with another friend
    2) User can win with all possible game conditions

### Technologies Used
    1) React
    2) JavaScript
    3) Bootstrap
    4) CSS

### Preview

    1) Preview displays all 3 conditions
        - Player 1 (X) wins
        - Player 2 (O) wins
        - Draw
